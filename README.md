# Minute-to-Pay Calculator

A lightweight, responsive web application designed for workers who track their time by the minute. Enter your minutes, view your exact earnings instantly, and log your shifts over time.

## 🚀 Features
* **Live Calculation:** Updates your estimated pay instantly with every keystroke.
* **Custom Hourly Rates:** Easily adjust your base rate for raises, different jobs, or overtime.
* **Persistent Shift History:** Save your completed shifts to a running log that remembers your data even if you refresh or close the browser.
* **Earnings Tracker:** Displays an accumulated total of your earnings across all saved shifts.
* **Mobile Friendly:** Clean, responsive user interface optimized for both desktop and mobile screens.

## 🛠️ Tech Stack
* HTML5
* CSS3 (Flexbox/Grid)
* Vanilla JavaScript (LocalStorage API)

## 📦 Installation & Usage
No installation required! 
1. Download or copy the `pay_calculator.html` file.
2. Double-click the file to open it instantly in any web browser on your phone or computer.
